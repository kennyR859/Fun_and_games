## 🐬 Games

### 🌊 [Luna's Ocean Math Adventure](./luna-ocean-math/)

An elementary math game teaching **5's complement** and **10's complement** addition through a fun dolphin-and-ocean theme.

**Play now:** https://kennyr859.github.io/Fun_and_games/luna-ocean-math/

**Features:**
- 🖐️ **Build a 5** — find the pair that makes 5 (1+4, 2+3, ...)
- 🙌 **Build a 10** — find the pair that makes 10 (1+9, 2+8, ...)
- 🐋 **Triple Dolphin** — add three numbers using the 5's complement trick, with auto explanations
- 🎯 **Pair Match** — memory game where matching cards must sum to 10
- ⭐ Score + ❤️ lives + 🏆 3-star rating system
- 🎊 Confetti, Web Audio sound effects, bubble animation
- 📱 Fully responsive (desktop, tablet, mobile)

**Built with:** Pure HTML + CSS + JavaScript (no frameworks, no dependencies).

## 📚 Learning Goals

The first game targets these Common Core standards:

- **CCSS.MATH.CONTENT.1.NBT.B.2** — Understanding base-10 and digit place value
- **CCSS.MATH.CONTENT.1.OA.C.6** — Mental-math strategies (make-a-five, make-a-ten) to add within 20

## 🛠️ Adding a New Game

1. Create a new folder at the repo root (e.g., `./my-new-game/`)
2. Drop a self-contained `index.html` inside
3. Add a card to this README
4. Push to `main` — GitHub Pages auto-deploys
